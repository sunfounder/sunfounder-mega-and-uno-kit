光敏电阻
==============

.. image:: img/photoresistor.png
    :width: 200
    :align: center

光敏电阻或光电管是一种光控可变电阻器。光敏电阻的电阻随着入射光强度的增加而降低；换句话说，它表现出光导性。

光敏电阻器可应用于光敏检测器电路和光激活和暗激活开关电路中作为电阻半导体。在黑暗中，光敏电阻的电阻可高达几兆欧 (MΩ)，而在光照下，光敏电阻的电阻可低至几百欧姆。
