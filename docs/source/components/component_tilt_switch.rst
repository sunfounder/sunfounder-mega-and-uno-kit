倾斜开关
=============================

.. image:: img/tilt_switch.png
    :width: 80
    :align: center

这里使用的倾斜开关是里面有一个金属球。它用于检测小角度的倾斜度。

原理很简单。当开关倾斜​​一定角度时，里面的小球滚下来，接触到外面的引脚所连接的两个触点，从而触发电路。否则球将远离触点，从而断开电路。

    :width: 600
